The Parallel Machine Scheduling Problem with Different Speeds and Release Times in the Ore Hauling Operation
Overview

This repository provides scripts and instructions for reading and processing data instances related to the research on the parallel machine scheduling problem with different speeds and release times in the ore hauling operation.


Requirements
Python 3.8+
Required Python packages listed in requirements.txt

Data Example
m (loading equipment)
v (hauling equipment)
p (production target)
t (horizon time)
